# Exercise 10: PDF Parser and RAG
## Tech Stack
- Node.js (Server)
- Frontend Framework (Client)

## System Dependencies
- **Node.js**: Required to install packages (`npm install`) and run the server/client without errors.
- **Python 3**: Required if you wish to run the `test_e2e.py` end-to-end integration test script.
- **Environment Variables**: An `OPENAI_API_KEY` (or equivalent) must be configured in the `.env` file of the server directory to run the AI features without errors.

## How to use
1. Make sure Node.js is installed on your system.
2. Navigate to `server`, configure the `.env` file, run `npm install`, then start the backend.
3. Navigate to `client`, run `npm install`, then start the frontend.
4. Upload `sample.pdf` to test parsing functionality.
